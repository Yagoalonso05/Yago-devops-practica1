# Tienda_online - Práctica 1.3
Proyecto de ejemplo para la Práctica 1.3: sistema de tienda online usando POO en Python.

Estructura:
- models/: clases Producto, Usuario, Pedido
- Services/: TiendaService para gestionar usuarios, productos y pedidos
- main.py: script de prueba que crea usuarios/productos y simula pedidos.

Cada archivo contiene comentarios explicativos en español en forma de '#'.
